# CommandPalette\n\n커맨드 팔레트는 ⌘K 스타일의 빠른 검색 및 네비게이션 UI입니다. 사용자가 검색어를 입력하면 명령어와 파일을 필터링합니다.\n\n## 언제 쓰나\n\n- **글로벌 검색**: 전체 앱에서 항목 검색\n- **빠른 네비게이션**: 사이드바 없이 페이지 이동\n- **명령어 실행**: 작업, 설정, 정렬 등 실행\n- **파일/항목 찾기**: 작업, 메시지, 문서 검색\n\n## 언제 쓰면 안 되나\n\n- **드롭다운 메뉴**: 일반 dropdown 사용\n- **페이지 검색**: `<input type=\"search\">` 사용\n- **필터 UI**: 고급 필터는 별도 패널\n\n## Props\n\n| 이름 | 타입 | 기본값 | 설명 |\n|------|------|--------|------|\n| `open` | `boolean` | — | 팔레트 표시 여부 |\n| `onClose` | `() => void` | — | 닫기 핸들러 |\n| `groups` | `CommandGroup[]` | — | 명령어 그룹 |\n| `onRun` | `(item: CommandItem) => void` | — | 항목 선택 핸들러 |\n| `placeholder` | `string` | \"명령 또는 파일 검색...\" | 검색 placeholder |\n\n### CommandGroup\n\n```tsx\ninterface CommandGroup {\n  label: string           // \"일반\", \"네비게이션\" 등\n  items: CommandItem[]    // 그룹의 명령어 목록\n}\n```\n\n### CommandItem\n\n```tsx\ninterface CommandItem {\n  id: string              // 유니크 ID\n  label: string           // 표시 라벨\n  icon?: ReactNode        // 선택 아이콘\n  hint?: string           // 오른쪽 힌트 (단축키 등)\n  keywords?: string       // 추가 검색 키워드\n}\n```\n\n## 기본 사용\n\n```tsx\nimport CommandPalette from '@/components/common/ds/CommandPalette'\nimport { useState } from 'react'\nimport { Plus, FileText, Settings } from 'lucide-react'\n\nfunction App() {\n  const [open, setOpen] = useState(false)\n  \n  // 글로벌 단축키 바인딩\n  useEffect(() => {\n    const handleKeyDown = (e: KeyboardEvent) => {\n      if ((e.metaKey || e.ctrlKey) && e.key === 'k') {\n        e.preventDefault()\n        setOpen(true)\n      }\n    }\n    window.addEventListener('keydown', handleKeyDown)\n    return () => window.removeEventListener('keydown', handleKeyDown)\n  }, [])\n  \n  const commands: CommandGroup[] = [\n    {\n      label: '작업',\n      items: [\n        { id: 'new-task', label: '새 작업', icon: <Plus size={12} /> },\n        { id: 'my-tasks', label: '내 작업', icon: <FileText size={12} /> }\n      ]\n    },\n    {\n      label: '설정',\n      items: [\n        { id: 'settings', label: '설정', icon: <Settings size={12} /> }\n      ]\n    }\n  ]\n  \n  const handleRun = (item: CommandItem) => {\n    switch (item.id) {\n      case 'new-task':\n        createTask()\n        break\n      case 'my-tasks':\n        navigateTo('/tasks')\n        break\n      case 'settings':\n        navigateTo('/settings')\n        break\n    }\n  }\n  \n  return (\n    <>\n      <Button onClick={() => setOpen(true)}>⌘K</Button>\n      <CommandPalette\n        open={open}\n        onClose={() => setOpen(false)}\n        groups={commands}\n        onRun={handleRun}\n      />\n    </>\n  )\n}\n```\n\n## 실제 사용 예\n\n### 작업 검색\n\n```tsx\nfunction TaskCommandPalette() {\n  const [open, setOpen] = useState(false)\n  const tasks = useTasks()\n
