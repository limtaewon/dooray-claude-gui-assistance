# Kbd\n\nKbd는 키보드 단축키를 표시하는 컴포넌트입니다. 모노스페이스 글꼴과 테두리로 물리 키처럼 보입니다.\n\n## 언제 쓰나\n\n- **단축키 표시**: 메뉴, 도움말, 커맨드 팔레트\n- **키 조합**: \"⌘ + K\", \"Ctrl + S\"\n- **지침**: \"Enter로 확인\"\n\n## 언제 쓰면 안 되나\n\n- **일반 텍스트**: 설명은 `<p>` 사용\n- **코드 블록**: `<code>` 또는 `<pre>` 사용\n\n## Props\n\n| 이름 | 타입 | 기본값 | 설명 |\n|------|------|--------|------|\n| `children` | `ReactNode` | — | 키 표시 |\n| `className` | `string` | — | 추가 CSS 클래스 |\n\n## 기본 사용\n\n```tsx\nimport Kbd from '@/components/common/ds/Kbd'\n\n// 단일 키\n<Kbd>Esc</Kbd>\n<Kbd>Enter</Kbd>\n<Kbd>Tab</Kbd>\n\n// 수정 키\n<Kbd>⌘</Kbd>\n<Kbd>Ctrl</Kbd>\n<Kbd>Shift</Kbd>\n<Kbd>Alt</Kbd>\n\n// 문자/숫자\n<Kbd>A</Kbd>\n<Kbd>1</Kbd>\n```\n\n## 스타일\n\n- 높이: 17px\n- padding: 0 4px\n- font-family: monospace\n- border: 1px + 2px 아래 (입체감)\n- 백그라운드: `--bg-surface`\n- 색: `--text-secondary`\n\n## 단축키 표시 패턴\n\n### 단일 키\n\n```tsx\nfunction HelpText() {\n  return (\n    <p>\n      <Kbd>Esc</Kbd>를 눌러 닫기\n    </p>\n  )\n}\n```\n\n### 키 조합\n\n```tsx\nfunction ShortcutDisplay() {\n  return (\n    <div style={{ display: 'inline-flex', gap: 4 }}>\n      <Kbd>⌘</Kbd>\n      <Kbd>K</Kbd>\n    </div>\n  )\n}\n\n// 또는 슬래시로 구분\n<>\n  <Kbd>Cmd</Kbd> + <Kbd>K</Kbd>\n</>\n```\n\n### 방향키\n\n```tsx\nfunction NavigationHint() {\n  return (\n    <span>\n      <Kbd>↑</Kbd> <Kbd>↓</Kbd>로 이동, <Kbd>↵</Kbd>로 실행\n    </span>\n  )\n}\n```\n\n## 커맨드 팔레트 사용\n\n```tsx\nfunction CommandPaletteHelp() {\n  return (\n    <div style={{ padding: 10 }}>\n      <h3>단축키</h3>\n      <table>\n        <tr>\n          <td><Kbd>↑</Kbd> <Kbd>↓</Kbd></td>\n          <td>이동</td>\n        </tr>\n        <tr>\n          <td><Kbd>↵</Kbd></td>\n          <td>실행</td>\n        </tr>\n        <tr>\n          <td><Kbd>Esc</Kbd></td>\n          <td>닫기</td>\n        </tr>\n      </table>\n    </div>\n  )\n}\n```\n\n## 메뉴 힌트\n\n```tsx\nfunction MenuWithHints() {\n  return (\n    <div className=\"ds-menu\">\n      <button className=\"ds-menu-item\">\n        저장\n        <span className=\"mi-hint\"><Kbd>⌘</Kbd><Kbd>S</Kbd></span>\n      </button>\n      <button className=\"ds-menu-item\">\n        실행 취소\n        <span className=\"mi-hint\"><Kbd>⌘</Kbd><Kbd>Z</Kbd></span>\n      </button>\n    </div>\n  )\n}\n```\n\n## 플랫폼별 수정 키\n\n```tsx\nfunction PlatformShortcut() {\n  const isMac = /Mac/.test(navigator.platform)\n  const modifier = isMac ? '⌘' : 'Ctrl'\n  \n  return (\n    <>\n      <Kbd>{modifier}</Kbd> + <Kbd>K</Kbd>\n    </>\n  )\n}\n```\n\n## 접근성\n\n```tsx\n// 좋음: aria-label로 의미 추가\n<div aria-label=\"커맨드 팔레트 단축키\">\n  <Kbd>⌘</Kbd> + <Kbd>K</Kbd>\n</div>\n\n// 좋음: 텍스트 라벨과 함께\n<p>\n  <Kbd>Esc</Kbd>를 눌러 대화상자를 닫을 수 있습니다.\n</p>\n```\n\n## 관련 토큰\n\n- `--bg-surface`: 배경\n- `--text-secondary`: 텍스트 색\n- `--bg-border`: 테두리\n- `--font-mono`: monospace 글꼴\n\n## 자주 하는 실수\n\n### ✗ 너무 큼\n\n```tsx\n// 안 됨 - 너무 큼\n<span style={{ fontSize: 14 }}>\n  <Kbd>A</Kbd>\n</span>\n\n// 좋음 - 작은 크기 유지\n<Kbd>A</Kbd>\n```\n\n### ✗ 많은 키 조합\n\n```tsx\n// 안 됨\n<Kbd>⌘</Kbd><Kbd>Shift</Kbd><Kbd>Alt</Kbd><Kbd>K</Kbd>\n\n// 좋음 - 최대 3개 이내\n<Kbd>⌘</Kbd><Kbd>Shift</Kbd><Kbd>K</Kbd>\n```\n\n## 예제 모음\n\n### 도움말 섹션\n\n```tsx\nfunction KeyboardShortcuts() {\n  return (\n    <div style={{ padding: 12 }}>\n      <h3>키보드 단축키</h3>\n      <div style={{ marginTop: 8 }}>\n        <div style={{ marginBottom: 6 }}>\n          <div style={{ display: 'inline-flex', gap: 4 }}>\n            <Kbd>⌘</Kbd> <Kbd>K</Kbd>\n          </div>\n          <span style={{ marginLeft: 8 }}>커맨드 팔레트</span>\n        </div>\n        <div style={{ marginBottom: 6 }}>\n          <div style={{ display: 'inline-flex', gap: 4 }}>\n            <Kbd>⌘</Kbd> <Kbd>P</Kbd>\n          </div>\n          <span style={{ marginLeft: 8 }}>프로젝트 검색</span>\n        </div>\n        <div>\n          <Kbd>Esc</Kbd>\n          <span style={{ marginLeft: 8 }}>팝업 닫기</span>\n        </div>\n      </div>\n    </div>\n  )\n}\n```\n\n### 입력 팁\n\n```tsx\nfunction InputWithHint() {\n  return (\n    <div>\n      <label>\n        검색\n        <small style={{ marginLeft: 4, color: 'var(--text-tertiary)' }}>\n          (<Kbd>Ctrl</Kbd>+<Kbd>K</Kbd> 단축키)\n        </small>\n      </label>\n      <Input placeholder=\"검색...\" />\n    </div>\n  )\n}\n```\n
