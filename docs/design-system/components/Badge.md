# Badge\n\n뱃지는 알림 숫자(notification count)나 작은 상태 표시를 나타내는 극소형 컴포넌트입니다. 보통 다른 요소 위에 겹쳐 표시됩니다.\n\n## 언제 쓰나\n\n- **알림 숫자**: 읽지 않은 메시지 수, 작업 수\n- **상태 점**: 온라인 상태\n- **카운터**: 사이드바 항목의 수\n\n## 언제 쓰면 안 되나\n\n- **일반 라벨**: `Chip` 사용\n- **큰 수**: 999+ 이상은 \"9+\" 또는 생략\n- **설명**: 뱃지는 숫자만 표시\n\n## Props\n\n| 이름 | 타입 | 기본값 | 설명 |\n|------|------|--------|------|\n| `children` | `ReactNode` | — | 뱃지 내용 (숫자, 아이콘) |\n| `tone` | `'orange' \\| 'blue' \\| 'emerald' \\| 'red' \\| 'violet'` | `'orange'` | 배경 색상 |\n\n## 기본 사용\n\n```tsx\nimport Badge from '@/components/common/ds/Badge'\n\n// 기본 (주황)\n<Badge>5</Badge>\n\n// 다른 색상\n<Badge tone=\"blue\">3</Badge>\n<Badge tone=\"emerald\">12</Badge>\n<Badge tone=\"red\">!</Badge>\n```\n\n## 크기\n\n뱃지는 자동으로 내용에 맞춰 크기 조정:\n\n- 높이: 16px\n- padding: 0 5px\n- font-size: 9px\n- font-weight: 700\n\n```tsx\n// 작은 숫자\n<Badge>1</Badge>        {/* 작음 */}\n<Badge>99</Badge>       {/* 조금 더 큼 */}\n<Badge>999</Badge>      {/* 더 크게 */}\n```\n\n## 색상 톤\n\n### Orange (기본, 강조)\n\n```tsx\n<Badge tone=\"orange\">5</Badge>\n```\n\n**용도**: 기본 알림 (두레이 브랜드 색)\n\n### Blue (정보)\n\n```tsx\n<Badge tone=\"blue\">3</Badge>\n```\n\n**용도**: 정보성 카운트\n\n### Emerald (성공)\n\n```tsx\n<Badge tone=\"emerald\">12</Badge>\n```\n\n**용도**: 완료, 성공 카운트\n\n### Red (긴급)\n\n```tsx\n<Badge tone=\"red\">!</Badge>\n```\n\n**용도**: 오류, 긴급 알림\n\n### Violet (특수)\n\n```tsx\n<Badge tone=\"violet\">@</Badge>\n```\n\n**용도**: 멘션 알림\n\n## 사용 패턴\n\n### 사이드바 항목\n\n```tsx\nfunction SidebarItem({ label, count }: { label: string;
