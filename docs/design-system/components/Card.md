# Card

카드는 콘텐츠를 독립적인 컨테이너로 감싸는 컴포넌트입니다. 균일한 스타일과 간격을 보장하여 복잡한 레이아웃을 정렬합니다.\n\n## 언제 쓰나\n\n- **데이터 카드**: 작업, 메시지, 이벤트의 요약 표시\n- **폼 섹션**: 관련된 입력 필드 그룹\n- **패널**: 사이드바의 설정, 필터, 정보 섹션\n- **타일**: 대시보드의 통계, 상태 요약\n\n## 언제 쓰면 안 되나\n\n- **버튼 대신 사용**: 클릭 가능한 요소는 `<button>` 사용\n- **레이아웃 틀**: 카드는 컨테이너지, 레이아웃 그리드 아님\n- **모달 대신**: 모달이 필요하면 `<Modal>` 사용\n\n## Props\n\n| 이름 | 타입 | 기본값 | 설명 |\n|------|------|--------|------|\n| `variant` | `'default' \\| 'raised' \\| 'flat'` | `'default'` | 스타일 |\n| `children` | `ReactNode` | — | 카드 내용 |\n| `className` | `string` | — | 추가 CSS 클래스 |\n| `...rest` | `HTMLAttributes<HTMLDivElement>` | — | 표준 div 속성 |\n\n## Variant 가이드\n\n### Default (표준)\n\n```tsx\nimport Card from '@/components/common/ds/Card'\n\n<Card variant=\"default\">\n  <h3>작업 제목</h3>\n  <p>작업 설명</p>\n</Card>\n```\n\n**특징**:\n- 배경: `--bg-surface` (표준)\n- 테두리: 1px `--bg-border`\n- 모서리: 8px radius\n\n**용도**: 일반적인 콘텐츠 컨테이너\n\n### Raised (돌출)\n\n```tsx\n<Card variant=\"raised\">\n  <h3>중요한 정보</h3>\n  <p>이 카드는 배경에서 돌출됩니다</p>\n</Card>\n```\n\n**특징**:\n- 배경: `--bg-surface-raised` (더 밝음)\n- 그림자: `var(--elev-1)` (가볍게 떠있음)\n- elevation으로 시각적 계층 표현\n\n**용도**: 중요한 정보, 강조할 콘텐츠\n\n### Flat (평평)\n\n```tsx\n<Card variant=\"flat\">\n  <span>배경 없는 카드</span>\n</Card>\n```\n\n**특징**:\n- 배경 없음 (transparent)\n- 테두리만 표시\n- 경량 스타일\n\n**용도**: 목록 아이템의 시각적 구분\n\n## 기본 사용\n\n### 작업 카드\n\n```tsx\nfunction TaskCard({ task }: { task: Task }) {\n  return (\n    <Card variant=\"default\">\n      <div style={{ padding: 10 }}>\n        <h3 style={{ marginBottom: 4 }}>{task.title}</h3>\n        <p style={{ fontSize: 12, color: 'var(--text-secondary)' }}>\n          {task.description}\n        </p>\n        <div style={{ marginTop: 8, display: 'flex', gap: 4 }}>\n          <Chip tone=\"blue\">{task.status}</Chip>\n          <Chip tone=\"orange\">{task.assignee}</Chip>\n        </div>\n      </div>\n    </Card>\n  )\n}\n```\n\n### 설정 패널\n\n```tsx\nfunction SettingsPanel() {\n  return (\n    <Card variant=\"default\">\n      <div style={{ padding: 12 }}>\n        <h4 className=\"text-section\">테마 설정</h4>\n        <div style={{ marginTop: 10 }}>\n          <label>\n            <input type=\"radio\" name=\"theme\" value=\"light\" /> 라이트\n          </label>\n          <label>\n            <input type=\"radio\" name=\"theme\" value=\"dark\" /> 다크\n          </label>\n        </div>\n      </div>\n    </Card>\n  )\n}\n```\n\n### 대시보드 타일\n\n```tsx\nfunction StatsTile({ label, value, change }: { label: string; value: number; change: number }) {\n  return (\n    <Card variant=\"raised\">\n      <div style={{ padding: 16, textAlign: 'center' }}>\n        <div style={{ fontSize: 10, color: 'var(--text-tertiary)' }}>\n          {label}\n        </div>\n        <div className=\"num-lg\" style={{ marginTop: 4 }}>\n          {value}\n        </div>\n        <div\n          style={{\n            fontSize: 11,\n            marginTop: 4,\n            color: change > 0 ? 'var(--c-emerald-fg)' : 'var(--c-red-fg)'\n          }}\n        >\n          {change > 0 ? '+' : ''}{change}%\n        </div>\n      </div>\n    </Card>\n  )\n}\n```\n\n## 간격 및 구조\n\n카드 내부는 직접 padding을 설정합니다:\n\n```tsx\n// 기본 padding\n<Card style={{ padding: 12 }}>\n  {/* 콘텐츠 */}\n</Card>\n\n// 여러 섹션\n<Card>\n  <div style={{ padding: '12px 10px', borderBottom: '1px solid var(--bg-border)' }}>\n    <h3>헤더</h3>\n  </div>\n  <div style={{ padding: 12 }}>\n    {/* 바디 */}\n  </div>\n  <div style={{ padding: 10, background: 'var(--bg-base)' }}>\n    {/* 푸터 */}\n  </div>\n</Card>\n```\n\n## 클릭 가능한 카드\n\n카드를 클릭 가능하게 하려면 `<button>` 래퍼를 사용하거나 onClick 핸들러를 추가합니다:\n\n```tsx\nfunction ClickableCard({ item, onClick }: { item: Item; onClick: () => void }) {\n  return (\n    <Card\n      as=\"button\"\n      onClick={onClick}\n      style={{\n        cursor: 'pointer',\n        transition: 'box-shadow 0.2s, transform 0.2s',\n        border: 'none',\n        textAlign: 'left'\n      }}\n      onMouseEnter={(e) => {\n        e.currentTarget.style.boxShadow = 'var(--elev-2)'\n        e.currentTarget.style.transform = 'translateY(-2px)'\n      }}\n      onMouseLeave={(e) => {\n        e.currentTarget.style.boxShadow = ''\n        e.currentTarget.style.transform = 'none'\n      }}\n    >\n      {item.title}\n    </Card>\n  )\n}\n```\n\n## 반응형 레이아웃\n\n```tsx\nfunction CardGrid({ items }: { items: Item[] }) {\n  return (\n    <div\n      style={{\n        display: 'grid',\n        gridTemplateColumns: 'repeat(auto-fill, minmax(300px, 1fr))',\n        gap: 12\n      }}\n    >\n      {items.map((item) => (\n        <Card key={item.id} variant=\"default\">\n          <div style={{ padding: 12 }}>\n            {item.title}\n          </div>\n        </Card>\n      ))}\n    </div>\n  )\n}\n```\n\n## 관련 토큰\n\n- `--bg-surface`: 기본 배경\n- `--bg-surface-raised`: 돌출 배경\n- `--bg-border`: 테두리 색\n- `--radius-md`: 모서리 반경 (8px)\n- `--elev-1`: 돌출 그림자\n\n## 자주 하는 실수\n\n### ✗ 과도한 padding/margin\n\n```tsx\n// 안 됨\n<Card style={{ padding: 40, margin: 20 }}>\n  {/* 너무 커짐 */}\n</Card>\n\n// 좋음\n<Card style={{ padding: 12 }}>\n  {/* 적절한 간격 */}\n</Card>\n```\n\n### ✗ 중첩된 카드\n\n```tsx\n// 안 됨 - 복잡함\n<Card>\n  <Card>\n    <Card>\n      {/* 카드 지옥 */}\n    </Card>\n  </Card>\n</Card>\n\n// 좋음 - 단순 계층\n<Card>\n  <div>{/* 섹션 */}</div>\n  <div>{/* 섹션 */}</div>\n</Card>\n```\n\n## 예제 모음\n\n### 메시지 카드\n\n```tsx\nfunction MessageCard({ msg }: { msg: Message }) {\n  return (\n    <Card variant=\"default\">\n      <div style={{ padding: 10 }}>\n        <div style={{ display: 'flex', gap: 8, marginBottom: 6 }}>\n          <Avatar name={msg.author} />\n          <div style={{ flex: 1 }}>\n            <strong>{msg.author}</strong>\n            <TimeAgo date={msg.createdAt} />\n          </div>\n        </div>\n        <p>{msg.text}</p>\n      </div>\n    </Card>\n  )\n}\n```\n\n### 폼 섹션\n\n```tsx\nfunction FormSection() {\n  return (\n    <Card variant=\"flat\">\n      <div style={{ padding: 12 }}>\n        <h3 className=\"text-section\">개인 정보</h3>\n        <div style={{ marginTop: 12, display: 'flex', flexDirection: 'column', gap: 10 }}>\n          <Input placeholder=\"이름\" />\n          <Input placeholder=\"이메일\" />\n        </div>\n      </div>\n    </Card>\n  )\n}\n```\n
