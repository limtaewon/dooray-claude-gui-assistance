# SegTabs\n\nSegTabs는 세그먼티드 탭 컴포넌트입니다. 단일 선택 옵션을 수평으로 표시하며, 설정, 필터, 뷰 전환에 사용됩니다.\n\n## 언제 쓰나\n\n- **뷰 전환**: \"목록\", \"그리드\", \"테이블\" 보기\n- **설정 섹션**: 탭으로 구분된 설정\n- **필터 활성화**: 현재 활성 필터 표시\n- **모드 선택**: \"라이트\", \"다크\" 모드\n\n## 언제 쓰면 안 되나\n\n- **많은 옵션** (5개 이상): 다른 UI 사용 (dropdown, list)\n- **독립적 토글**: checkbox/radio 사용\n- **다중 선택**: checkbox 그룹 사용\n\n## Props\n\n| 이름 | 타입 | 기본값 | 설명 |\n|------|------|--------|------|\n| `items` | `SegTabItem<T>[]` | — | 탭 목록 |\n| `value` | `T` | — | 현재 선택 키 |\n| `onChange` | `(key: T) => void` | — | 선택 변경 핸들러 |\n| `className` | `string` | — | 추가 CSS 클래스 |\n\n**SegTabItem**:\n```tsx\ninterface SegTabItem<T extends string = string> {\n  key: T                  // 유니크 키\n  label: ReactNode        // 표시 라벨\n  icon?: ReactNode        // 선택사항 아이콘\n}\n```\n\n## 기본 사용\n\n```tsx\nimport SegTabs from '@/components/common/ds/SegTabs'\nimport { useState } from 'react'\n\nfunction ViewToggle() {\n  const [view, setView] = useState<'list' | 'grid'>('list')\n  \n  return (\n    <SegTabs\n      items={[\n        { key: 'list', label: '목록' },\n        { key: 'grid', label: '그리드' }\n      ]}\n      value={view}\n      onChange={setView}\n    />\n  )\n}\n```\n\n## 아이콘 포함\n\n```tsx\nimport { List, Grid3x3, Table } from 'lucide-react'\n\nfunction ViewSelector() {\n  const [view, setView] = useState<'list' | 'grid' | 'table'>('list')\n  \n
