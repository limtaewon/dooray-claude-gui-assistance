# TimeAgo\n\nTimeAgo는 시간을 상대적(\"3분 전\") 또는 절대적(\"오후 2:30\") 형식으로 표시하는 컴포넌트입니다.\n\n## 언제 쓰나\n\n- **메시지/댓글**: \"5분 전\"\n- **작업 타임스탬프**: \"2시간 전 생성됨\"\n- **로그 시간**: 상대/절대 시간 표시\n- **업데이트 표시**: \"방금\", \"1일 전\"\n\n## 언제 쓰면 안 되나\n\n- **정확한 시간 필요**: 절대 시간 명시 필요 시 추가 텍스트\n- **날짜만**: 날짜 포맷팅은 별도 함수\n\n## Props\n\n| 이름 | 타입 | 기본값 | 설명 |\n|------|------|--------|------|\n| `date` | `Date \\| string \\| number` | — | 시간 (Date, ISO string, timestamp) |\n| `absolute` | `boolean` | `false` | true면 절대시간, false면 상대시간 |\n| `updateInterval` | `number` | `60000` | 상대시간 갱신 주기(ms) |\n\n## 기본 사용\n\n```tsx\nimport TimeAgo from '@/components/common/ds/TimeAgo'\n\n// 상대 시간 (기본)\n<TimeAgo date={new Date()} />  // \"방금\"\n<TimeAgo date={Date.now() - 5 * 60 * 1000} />  // \"5분 전\"\n<TimeAgo date=\"2026-05-13T10:30:00Z\" />  // 상대 시간\n\n// 절대 시간\n<TimeAgo date={new Date()} absolute={true} />  // \"오후 2:30\"\n```\n\n## 상대 시간 포맷\n\n```tsx\n// < 5초: \"방금\"\n<TimeAgo date={Date.now()} />\n\n// < 60초: \"3초 전\"\n<TimeAgo date={Date.now() - 3000} />\n\n// < 1시간: \"15분 전\"\n<TimeAgo date={Date.now() - 15 * 60 * 1000} />\n\n// < 1일: \"2시간 전\"\n<TimeAgo date={Date.now() - 2 * 60 * 60 * 1000} />\n\n// < 1주: \"3일 전\"\n<TimeAgo date={Date.now() - 3 * 24 * 60 * 60 * 1000} />\n\n// >= 1주: \"03-13\" (날짜)\n<TimeAgo date=\"2026-03-13T10:30:00Z\" />\n```\n\n## 절대 시간 포맷\n\n```tsx\n// 형식: \"MM-DD 오후 hh:mm\"\n<TimeAgo date={new Date()} absolute={true} />\n// 예: \"05-13 오후 2:30\"\n```\n\n## 자동 갱신\n\n상대 시간은 지정된 주기로 자동 갱신됩니다:\n\n```tsx\n// 60초마다 갱신 (기본)\n<TimeAgo date={messageDate} />\n\n// 1초마다 갱신 (실시간)\n<TimeAgo date={messageDate} updateInterval={1000} />\n\n// 갱신 안 함 (절대 시간)\n<TimeAgo date={messageDate} absolute={true} />\n```\n\n## 사용 패턴\n\n### 메시지 타임스탬프\n\n```tsx\nfunction
