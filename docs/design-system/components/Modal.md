# Modal\n\n모달은 포털을 통해 렌더링되는 다이얼로그 컴포넌트입니다. 사용자의 주의를 요구하는 중요한 콘텐츠를 표시합니다.\n\n## 언제 쓰나\n\n- **확인 다이얼로그**: \"정말 삭제하시겠습니까?\"\n- **입력 폼**: 데이터 생성/편집\n- **상세 정보**: 항목의 전체 정보 보기\n- **경고/알림**: 중요한 메시지\n\n## 언제 쓰면 안 되나\n\n- **간단한 알림**: `Toast` 사용\n- **메뉴**: `CommandPalette` 또는 dropdown 사용\n- **사이드 패널**: 모달이 아닌 슬라이드 패널 사용\n\n## Props\n\n| 이름 | 타입 | 기본값 | 설명 |\n|------|------|--------|------|\n| `open` | `boolean` | — | 모달 표시 여부 |\n| `onClose` | `() => void` | — | 닫기 핸들러 |\n| `title` | `ReactNode` | — | 헤더 제목 |\n| `icon` | `ReactNode` | — | 헤더 아이콘 |\n| `children` | `ReactNode` | — | 모달 본문 |\n| `footer` | `ReactNode` | — | 하단 버튼 영역 |\n| `width` | `number \\| string` | `460px` | 모달 너비 |\n| `dismissable` | `boolean` | `true` | ESC/backdrop 클릭으로 닫기 |\n| `resizable` | `boolean` | `false` | 우하단 드래그로 크기 조절 |\n\n## 기본 사용\n\n```tsx\nimport Modal from '@/components/common/ds/Modal'\nimport { useState } from 'react'\n\nfunction ModalExample() {\n  const [open, setOpen] = useState(false)\n  \n  return (\n    <>\n      <Button onClick={() => setOpen(true)}>모달 열기</Button>\n      \n      <Modal\n        open={open}\n        onClose={() => setOpen(false)}\n        title=\"알림\"\n        children={<p>모달 콘텐츠</p>}\n        footer={\n          <Button variant=\"primary\" onClick={() => setOpen(false)}>\n            확인\n          </Button>\n        }\n      />\n    </>\n  )\n}\n```\n\n## 아이콘 포함\n\n```tsx\nimport { AlertTriangle } from 'lucide-react'\n\n<Modal\n  open={open}\n  onClose={onClose}\n  title=\"삭제 확인\"\n  icon={<AlertTriangle size={16} color=\"var(--c-red-fg)\" />}\n>\n  <p>정말 삭제하시겠습니까? 이 작업은 되돌릴 수 없습니다.</p>\n</Modal>\n```\n\n## Footer 버튼 레이아웃\n\n```tsx\nfunction ConfirmModal() {\n  const [open, setOpen] = useState(false)\n  \n  return (\n    <Modal\n setOpen(false)}>\n            취소\n          </Button>\n          <Button variant=\"danger\" onClick={handleDelete}>\n            삭제\n          </Button>\n        </div>\n      }\n    >\n      정말 삭제하시겠습니까?\n    </Modal>\n  )\n}\n```\n\n## 폼 모달\n\n```tsx\nfunction CreateTaskModal() {\n  const [open, setOpen] = useState(false)\n  const [title, setTitle] = useState('')\n  const [desc, setDesc] = useState('')\n  \n  const handleCreate = () => {\n    // API 호출\n    setOpen(false)\n  }\n  \n  return (\n    <Modal\n      open={open}\n      onClose={() => setOpen(false)}\n      title=\"새 작업 생성\"\n      width={500}\n      footer={\n        <div style={{ display: 'flex', gap: 6 }}>\n          <Button variant=\"secondary\" onClick={() => setOpen(false)}>\n            취소\n          </Button>\n          <Button variant=\"primary\" onClick={handleCreate}>\n            생성\n          </Button>\n        </div>\n      }\n    >\n      <div style={{ display: 'flex', flexDirection: 'column', gap: 12 width={300} />\n\n// 넓은 (600px)\n<Modal width={600} />\n\n// 백분율\n<Modal width=\"80vw\" />\n```\n\n## 닫기 제어\n\n### Dismissable (기본)\n\n```tsx\n// ESC, backdrop 클릭으로 닫기 가능\n<Modal open={open} onClose={onClose} dismissable={true} />\n```\n\n### Non-dismissable\n\n```tsx\n// 중요한 작업 중 - 의도적으로 닫기만 가능\n<Modal\n  open={open}\n  onClose={onClose}\n  dismissable={false}\n  title=\"데이터 처리 중...\"\n  children={<div className=\"ds-spinner\" />}\n/>\n```\n\n## 크기 조절 가능 (Resizable)\n\n```tsx\n<Modal\n  open={open}\n  onClose={onClose}\n  title=\"상세 정보\"\n  resizable={true}\n  children={<LargeContent />}\n/>\n```\n\n**특징**:\n- 우하단 모서리 드래그로 크기 조절\n- 최소 크기: 360×140px\n- 최대 크기: 92vw × 92vh\n- 콘텐츠는 스크롤 가능\n\n## 스타일\n\n- 너비: 460px (기본)\n- 배경: `--bg-surface-raised`\n- 테두리: 1px `--bg-border`\n- 모서리: 10px\n- 그림자: `--elev-3` (높음)\n- 오버레이: `--overlay-bg` (40% 검정 또는 60% 검정)\n\n## 포털 렌더링\n\nModal은 `document.body` 포털로 렌더링되므로, z-index 충돌 걱정 없음:\n\n```tsx\n// Modal은 항상 최상단에 표시\n<div style={{ zIndex: 100 }}>\n  {/* 다른 콘텐츠 */}\n</div>\n<Modal open={open} />  {/* 위에 렌더링됨 */}\n```\n\n## 애니메이션\n\n```css\n/* backdrop: fadeIn 150ms */\n/* modal: slideUp + scale 180ms cubic-bezier */\n```\n\n- 부드러운 진입\n- 빠른 응답성\n- 모바일 친화적\n\n## 관련 토큰\n\n- `--bg-surface-raised`: 모달 배경\n- `--elev-3`: 그림자\n- `--overlay-bg`: 배경 오버레이\n- `--text-primary`: 제목\n- `--bg-border`: 테두리\n\n## 접근성\n\n```tsx\n// 좋음: 제목 명시\n<Modal\n  open={open}\n  title=\"작업 삭제 확인\"\n  />\n\n// 좋음: 포커스 관리 (ESC로 닫기)\n<Modal\n  open={open}\n  onClose={onClose}\n  dismissable={true}\n/>\n\n// 좋음: 명확한 버튼\n<Modal\n  footer={\n    <>\n      <Button onClick={() => setOpen(false)}>취소</Button>\n      <Button variant=\"primary\" onClick={handleConfirm}>\n        확인\n      </Button>\n    </>\n  }\n/>\n```\n\n## 자주 하는 실수\n\n### ✗ Footer onClick={() => setOpen(false)}>취소</Button>\n      <Button variant=\"danger\">삭제</Button>\n    </>\n  }\n>\n  정말 삭제하시겠습니까?\n</Modal>\n```\n\n### ✗ 너무 많은 콘텐츠\n\n```tsx\n// 안 됨 - 한 화면에 안 맞음\n<Modal
