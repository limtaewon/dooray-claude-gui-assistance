# Toast\n\n토스트는 우하단에 나타나는 임시 알림입니다. 모달과 달리 사용자의 작업을 방해하지 않으며, 자동 소멸합니다.\n\n## 언제 쓰나\n\n- **작업 완료**: \"저장되었습니다\"\n- **오류 알림**: \"네트워크 오류가 발생했습니다\"\n- **성공 피드백**: \"복사되었습니다\"\n- **AI 작업**: \"분석 시작했습니다\"\n\n## 언제 쓰면 안 되나\n\n- **중요한 확인**: `Modal` 사용\n- **사용자 입력 필요**: 모달 또는 입력 필드 사용\n- **영구 메시지**: 인라인 메시지 또는 배너 사용\n\n## 사용법\n\n### 전역 Provider 설정\n\nApp.tsx에서:\n\n```tsx\nimport ToastHost from '@/components/common/ds/Toast'\n\nfunction App() {\n  return (\n    <ToastHost>\n      {/* 앱 콘텐츠 */}\n    </ToastHost>\n  )\n}\n```\n\n### useToast 훅\n\n```tsx\nimport { useToast } from '@/components/common/ds/Toast'\n\nfunction MyComponent() {\n  const toast = useToast()\n  \n  return (\n    <Button onClick={() => toast.success('저장됨')}>저장</Button>\n  )\n}\n```\n\n## Props\n\n### useToast API\n\n| 메서드 | 시그니처 | 용도 |\n|--------|---------|------|\n| `push(input)` | `(t: ToastInput) => void` | 커스텀 토스트 |\n| `success(title, body?)` | `(title, body?) => void` | 성공 메시지 |\n| `error(title, body?)` | `(title, body?) => void` | 오류 메시지 |\n| `warn(title, body?)` | `(title, body?) => void` | 경고 메시지 |\n| `ai(title, body?)` | `(title, body?) => void` | AI 작업 메시지 |\n| `info(title, body?)` | `(title, body?) => void` | 정보 메시지 |\n\n### ToastInput\n\n```tsx\ninterface ToastInput {\n  title: ReactNode          // 필수: 제목\n  body?: ReactNode          // 선택: 본문 설명\n  tone?: ToastTone          // 선택: 'default' | 'success' | 'error' | 'warn' | 'ai'\n  duration?: number         // 선택: ms (기본 3600ms)\n}\n```\n\n## 톤(Style) 가이드\n\n### Success (초록)\n\n```tsx\nconst toast = useToast()\ntoast.success('저장되었습니다')\n```\n\n**색상**: `--c-emerald-solid`\n\n**용도**:\n- 작업 완료\n- 저장 성공\n- 업로드 완료\n\n### Error (빨강)\n\n```tsx\ntoast.error('오류 발생', '파일을 찾을 수 없습니다')\n```\n\n**색상**: `--c-red-solid`\n\n**용도**:\n- API 오류\n- 네트워크 실패\n- 검증 오류\n\n### Warn (노랑)\n\n```tsx\ntoast.warn('주의', '이 작업은 되돌릴 수 없습니다')\n```\n\n**색상**: `--c-yellow-solid`\n\n**용도**:\n- 경고 메시지\n- 주의 필요\n- 미리 알림\n\n### AI (주황)\n\n```tsx\ntoast.ai('분석 시작', 'Claude가 문서를 분석 중입니다')\n```\n\n**색상**: `--clover-orange`\n\n**용도**:\n- AI 작업 시작\n- Claude 연동\n- 자동화 작업\n\n### Default (파랑)\n\n```tsx\ntoast.info('안내', '새로운 메시지가 있습니다')\n```\n\n**색상**: `--clover-blue`\n\n**용도**:\n- 일반 정보\n- 상태 변경\n- 알림\n\n## 기본 사용 예\n\n### 저장\n\n```tsx\nfunction SaveButton() {\n  const toast = useToast()\n  \n  const handleSave = async () => {\n    try {\n      await saveData()\n      toast.success('저장되었습니다')\n    } catch (err) {\n      toast.error('저장 실패', err.message)\n    }\n  }\n  \n  return <Button onClick={handleSave}>저장</Button>\n}\n```\n\n### 복사\n\n```tsx\nfunction CopyButton({ text }: { text: string }) {\n  const toast = useToast()\n  \n  const handleCopy = () => {\n    navigator.clipboard.writeText(text)\n    toast.success('복사되었습니다')\n  }\n  \n  return <Button onClick={handleCopy}>복사</Button>\n}\n```\n\n### 파일 업로드\n\n```tsx\nfunction FileUpload() {\n  const toast = useToast()\n  \n  const handleUpload = async (file: File) => {\n    const formData = new FormData()\n    formData.append('file', file)\n    \n    toast.ai('업로드 중...', file.name)\n    \n    try {\n      await fetch('/api/upload', { method: 'POST', body: formData })\n      toast.success('업로드 완료')\n    } catch (err) {\n      toast.error('업로드 실패', err.message)\n    }\n  }\n  \n  return (\n    <input\n      type=\"file\"\n      onChange={(e) => e.target.files?.[0] && handleUpload(e.target.files[0])}\n    />\n  )\n}\n```\n\n## 커스텀 토스트\n\n```tsx\nconst toast = useToast()\n\ntoast.push({\n  title: '커스텀 토스트',\n  body: '이것은 커스텀 메시지입니다',\n  tone: 'success',\n  duration: 5000  // 5초 후 자동 소멸\n})\n```\n\n## 지속 시간\n\n```tsx\nconst toast = useToast()\n\n// 기본: 3600ms (약 3.6초)\ntoast.success('저장됨')\n\n// 더 오래 표시\ntoast.error('오류 발생', '이 메시지는 10초 동시 표시', {\n  duration: 10000\n})\n\n// 짧게\ntoast.push({\n  title: '빠른 알림',\n  duration: 1000  // 1초\n})\n```\n\n## 구조\n\n```\n┌─────────────────────────┐\n│ icon  title       close  │\n│       body               │\n└─────────────────────────┘\n```\n\n- **높이**: 가변 (제목: 12px, 본문: 11px)\n- **padding**: 8px 10px\n- **테두리**: 3px 왼쪽 (톤에 따라 색상)\n- **그림자**: `--elev-2`\n\n## 스타일\n\n```css\n.ds-toast {\n  background: var(--bg-surface-raised);\n  border: 1px solid var(--bg-border);\n  border-left: 3px solid var(--c-blue-solid);  /* 톤에 따라 변경 */\n}\n```\n\n## 배치\n\n```css\n.ds-toast-viewport {\n  position: fixed;\n  bottom: 14px;       /* 화면 하단에서 14px */\n  right: 14px;        /* 오른쪽에서 14px */\n  z-index: 80;        /* 높음 */\n}\n```\n\n## 애니메이션\n\n```css\n@keyframes ds-toastIn {\n  from { opacity: 0; transform: translateY(6px) scale(0.98); }\n  to   { opacity: 1; transform: translateY(0) scale(1); }\n}\n```\n\n진입: 220ms cubic-bezier(.2,.8,.2,1)\n\n## 관련 토큰\n\n- `--bg-surface-raised`: 배경\n- `--c-emerald-solid`: success 테두리\n- `--c-red-solid`: error 테두리\n- `--c-yellow-solid`: warn 테두리\n- `--clover-orange`: ai 테두리\n- `--clover-blue`: default 테두리\n- `--elev-2`: 그림자\n\n## 접근성\n\n```tsx\nconst toast = useToast()\n\n// 좋음: 제목만으로 의미 명확\ntoast.success('저장되었습니다')\n\n// 좋음: 상세 설명\ntoast.error(\n  '저장 실패',\n  '네트워크 연결을
