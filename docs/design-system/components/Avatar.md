# Avatar\n\n아바타는 사용자를 나타내는 원형 컴포넌트입니다. 프로필 이미지가 없을 때 이니셜과 해시 기반 색상을 자동 생성합니다.\n\n## 언제 쓰나\n\n- **사용자 표시**: 댓글, 메시지, 담당자\n- **작은 프로필**: 리스트 아이템, 헤더\n- **색상 코딩**: 사용자를 색으로 구분\n\n## 언제 쓰면 안 되나\n\n- **큰 프로필 사진**: 전체 프로필 페이지는 `<img>` 직접 사용\n- **아이콘 대체**: 상태 아이콘은 `Chip` 사용\n\n## Props\n\n| 이름 | 타입 | 기본값 | 설명 |\n|------|------|--------|------|\n| `name` | `string` | `''` | 사용자명 (첫 2글자 사용) |\n| `size` | `'sm' \\| 'md' \\| 'lg' \\| 'xl'` | `'md'` | 크기 |\n| `tone` | `string` | — | 색상 override (기본은 이름 해시) |\n| `className` | `string` | — | 추가 CSS 클래스 |\n\n## 크기 가이드\n\n| Size | 픽셀 | 용도 |\n|------|------|------|\n| `sm` | 18px | 작은 목록, 인라인 |\n| `md` | 22px | **기본** — 대부분의 UI |\n| `lg` | 28px | 컨테이너 헤더 |\n| `xl` | 36px | 프로필 요약 |\n\n## 기본 사용\n\n```tsx\nimport Avatar from '@/components/common/ds/Avatar'\n\n// 기본 (md)\n<Avatar name=\"Kim Chulsu\" />\n\n// 크기 제어\n<Avatar name=\"Kim Chulsu\" size=\"sm\" />\n<Avatar name=\"Kim Chulsu\" size=\"lg\" />\n\n// 색상 override (16진수 색)\n<Avatar name=\"Kim Chulsu\" tone=\"#FF5733\" />\n```\n\n## 색상 자동 선택 (해시 기반)\n\n이름을 기반으로 일관된 색이 자동 선택됩니다:\n\n```tsx\n// \"Kim Chulsu\" → 항상 같은 색\n<Avatar name=\"Kim Chulsu\" />\n<Avatar name=\"Kim Chulsu\" />  // 동일한 색\n\n// 다른 이름은 다른 색\n<Avatar name=\"Lee Minsu\" />    // 다른 색\n<Avatar name=\"Park Suho\" />    // 또 다른 색\n```\n\n**구현**: 이름 문자의 charCode 합을 팔레트 길이로 모듈로 연산하여 인덱스 결정.\n\n```tsx\nconst PALETTE = [\n  '#EA580C',  // 주황 (두레이)\n  '#2563EB',  // 파랑 (Claude)\n  '#22C55E',  // 초록\n  '#A78BFA',  // 보라\n  '#FACC15',  // 노랑\n  '#EF4444'   // 빨강\n]\n```\n\n## 사용 패턴\n\n### 메시지에서\n\n```tsx\nfunction MessageItem({ msg }: { msg: Message }) {\n  return (\n    <div style={{ display: 'flex', gap: 8 }}>\n      <Avatar name={msg.author} size=\"md\" />\n      <div style={{ flex: 1 }}>\n        <strong>{msg.author}</strong>\n        <p>{msg.text}</p>\n      </div>\n    </div>\n  )\n}\n```\n\n### 리스트 아이템\n\n```tsx\nfunction UserList({ users }: { users: User[] }) {\n  return (\n    <div>\n      {users.map((u) => (\n        <div key={u.id} style={{ display: 'flex', alignItems: 'center', gap: 8, padding: 8 }}>\n          <Avatar name={u.name} size=\"sm\" />\n          <span>{u.name}</span>\n        </div>\n      ))}\n    </div>\n  )\n}\n```\n\n### 담당자 표시\n\n```tsx\nfunction TaskAssignee({ task }: { task: Task }) {\n  return (\n    <div style={{ display: 'flex', alignItems: 'center', gap: 6 }}>\n      <Avatar name={task.assignee} size=\"sm\" />\n      <small>{task.assignee}</small>\n    </div>\n  )\n}\n```\n\n### 여러 사용자 (스택)\n\n```tsx\nfunction AvatarStack({ names }: { names: string[] }) {\n  return (\n    <div style={{ display: 'flex', marginLeft: -(8) }}>\n      {names.slice(0, 3).map((name, i) => (\n        <div\n          key={name}\n          style={{\n            marginLeft: i > 0 ? -8 : 0,\n            zIndex: names.length - i,\n            border: '2px solid var(--bg-surface)'\n          }}\n        >\n          <Avatar name={name} size=\"md\" />\n        </div>\n      ))}\n      {names.length > 3 && (\n        <div\n          style={{\n            marginLeft: -8,\n            zIndex: 0,\n            display: 'flex',\n            alignItems: 'center',\n            justifyContent: 'center',\n            width: 22,\n            height: 22,\n            borderRadius: 999,\n            background: 'var(--bg-surface)',\n            border: '2px solid var(--bg-border)',\n            fontSize: 9,\n            fontWeight: 600\n          }}\n        >\n          +{names.length - 3}\n        </div>\n      )}\n    </div>\n  )\n}\n```\n\n### 색상 override (팀 색상)\n\n팀별 색상을 고정하려면 override를 사용합니다:\n\n```tsx\nfunction TeamAvatar({ member }: { member: TeamMember }) {\n  const teamColors: Record<string, string> = {\n    'Engineering': '#3B82F6',\n    'Design': '#8B5CF6',\n    'Product': '#EC4899'\n  }\n  \n  return (\n    <Avatar\n      name={member.name}\n      tone={teamColors[member.team]}\n    />\n  )\n}\n```\n\n## 이니셜 추출 로직\n\n```tsx\n// \"Kim Chulsu\" → \"KI\"\n// \"K\" → \"K·\"\n// \"\" → \"·\"\nconst initials = name.trim().slice(0, 2).toUpperCase() || '·'\n```\n\n한글도 자동 지원:\n\n```tsx\n<Avatar name=\"김철수\" />  // \"김\"\n<Avatar name=\"이민수\" />  // \"이\"\n<Avatar name=\"박수호\" />  // \"박\"\n```\n\n## 색상 스타일\n\nAvatar는 inline style로 색을 적용합니다 (Tailwind 미사용):\n\n```tsx\nstyle={{\n  background: bg + '22',        // 배경색 22% alpha\n  color: bg,                     // 텍스트는 원본 색\n  borderColor: bg + '44'         // 테두리는 44% alpha\n}}\n```\n\n결과적으로 밝은 배경에 어두운 텍스트, 다크 모드에서도 읽을 수 있도록 설계되었습니다.\n\n## 관련 토큰\n\nAvatar는 고정 색상을 사용하지만, 참고용:\n- 파레트: 6가지 색 (두레이, Claude, 초록, 보라, 노랑, 빨강)\n- 배경 투명도: 13% (rgb에 22 추가)\n- 테두리 투명도: 27% (rgb에 44 추가)\n\n## 접근성\n\n```tsx\n// 좋음: 이미지 alt text 대체\n<div\n  role=\"img\"\n  aria-label=\"김철수 프로필\"\n>\n  <Avatar name=\"Kim Chulsu\" />\n</div>\n\n// 좋음: 리스트에서\n<div role=\"list\">\n  {users.map(u => (\n    <div key={u.id} role=\"listitem\">\n      <Avatar name={u.name} />\n      {u.name}\n    </div>\n  ))}\n</div>\n```\n\n## 자주 하는 실수\n\n### ✗ 긴 이름\n\n```tsx\n// 안 됨 - 이니셜이 제대로 안 나옴\n<Avatar name=\"김철수이민수박수호\" />\n\n// 좋음 - 짧은 이름\n<Avatar name=\"Kim
