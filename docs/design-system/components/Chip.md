# Chip

칩은 상태, 태그, 라벨, 카운트 등을 작고 간결하게 표시하는 컴포넌트입니다. 7가지 색상 톤을 지원합니다.\n\n## 언제 쓰나\n\n- **상태 표시**: 작업 상태(작업 중, 해결됨 등)\n- **태그/라벨**: 카테고리, 키워드\n- **필터**: 활성 필터 표시\n- **인라인 메타**: 우선순위, 담당자, 마감일\n\n## 언제 쓰면 안 되나\n\n- **제거 가능한 태그**: 태그 입력은 다른 라이브러리 사용 (`react-tag-input` 등)\n- **많은 텍스트**: 2줄 이상 금지\n- **액션 버튼**: 버튼이 필요하면 `<Button>` 사용\n\n## Props\n\n| 이름 | 타입 | 기본값 | 설명 |\n|------|------|--------|------|\n| `tone` | `'blue' \\| 'orange' \\| 'emerald' \\| 'red' \\| 'violet' \\| 'yellow' \\| 'neutral'` | `'neutral'` | 색상 톤 |\n| `dot` | `boolean` | `false` | 앞에 작은 점 표시 |\n| `square` | `boolean` | `false` | 모서리 사각형 (기본은 둥글게) |\n| `children` | `ReactNode` | — | 칩 텍스트 |\n| `className` | `string` | — | 추가 CSS 클래스 |\n\n## 색상 톤\n\n### Blue (정보, 작업 중)\n\n```tsx\nimport Chip from '@/components/common/ds/Chip'\n\n<Chip tone=\"blue\">작업 중</Chip>\n<Chip tone=\"blue\" dot>활성</Chip>\n```\n\n**색상**: `--c-blue-bg` (배경), `--c-blue-fg` (텍스트)\n\n### Orange (AI, 액션)\n\n```tsx\n<Chip tone=\"orange\">AI 기능</Chip>\n<Chip tone=\"orange\" square>두레이</Chip>\n```\n\n### Red (오류, 위험)\n\n```tsx\n<Chip tone=\"red\">오류</Chip>\n<Chip tone=\"red\" dot>주의</Chip>\n```\n\n### Emerald (성공, 완료)\n\n```tsx\n<Chip tone=\"emerald\">완료</Chip>\n<Chip tone=\"emerald\" dot>성공</Chip>\n```\n\n### Violet (멘션, 참고)\n\n```tsx\n<Chip tone=\"violet\">@멘션됨</Chip>\n<Chip tone=\"violet\" square>참고</Chip>\n```\n\n### Yellow (경고)\n\n```tsx\n<Chip tone=\"yellow\">경고</Chip>\n<Chip tone=\"yellow\" dot>주의</Chip>\n```\n\n### Neutral (기본, 중립)\n\n```tsx\n<Chip tone=\"neutral\">기본</Chip>\n<Chip tone=\"neutral\" dot>중립</Chip>\n```\n\n## 점(Dot) 표시\n\n칩 앞에 작은 점을 추가하여 상태 표시:\n\n```tsx\n// 온라인 상태\n<Chip tone=\"emerald\" dot>온라인</Chip>\n\n// 오프라인 상태\n<Chip tone=\"neutral\" dot>오프라인</Chip>\n\n// 작업 중\n<Chip tone=\"blue\" dot>작업 중</Chip>\n```\n\n## 모서리 스타일\n\n기본은 완전히 둥글게(`border-radius: 999px`), square 옵션으로 약간 사각형:\n\n```tsx\n// 둥글게 (기본)\n<Chip tone=\"blue\">라벨</Chip>\n\n// 사각형\n<Chip tone=\"blue\" square>라벨</Chip>\n```\n\n## 크기\n\n칩 크기는 고정이며, 텍스트 길이에 따라 자동 조정:\n\n- 높이: 18px\n- padding: 0 7px\n- font-size: 10px\n- 최소 너비: 없음 (텍스트에 맞춤)\n\n## 사용 패턴\n\n### 워크플로우 상태\n\n```tsx\nfunction TaskItem({ task }: { task: Task }) {\n  return (\n    <div>\n      <h3>{task.title}</h3>\n      {task.status === 'backlog' && <Chip tone=\"neutral\">백로그</Chip>}\n      {task.status === 'working' && <Chip tone=\"blue\" dot>작업 중</Chip>}\n      {task.status === 'resolved' && <Chip tone=\"emerald\" dot>해결됨</Chip>}\n      {task.status === 'closed' && <Chip tone=\"neutral\">종료</Chip>}\n      {task.isOverdue && <Chip tone=\"red\">기한 초과</Chip>}\n    </div>\n  )\n}\n```\n\n### 태그 목록\n\n```tsx\nfunction TagList({ tags }: { tags: string[] }) {\n  return (\n    <div style={{ display: 'flex', gap: 6, flexWrap: 'wrap' }}>\n      {tags.map((tag) => (\n        <Chip key={tag} tone=\"blue\">\n          {tag}\n        </Chip>\n      ))}\n    </div>\n  )\n}\n```\n\n### 필터 버튼 (선택 스타일)\n\n```tsx\nfunction FilterChips() {\n  const [selected, setSelected] = useState<string[]>([])\n  const filters = ['완료', '작업 중', '대기 중']\n  \n  return (\n    <div style={{ display: 'flex', gap: 6 }}>\n      {filters.map((f) => (\n        <button\n          key={f}\n          onClick={() => setSelected(s => \n            s.includes(f) ? s.filter(x => x !== f) : [...s, f]\n          )}\n          style={{\n            padding: '0 6px',\n            height: 18,\n            border: 'none',\n            borderRadius: 999,\n            background: selected.includes(f) ? 'var(--c-blue-bg)' : 'var(--bg-surface)',\n            color: selected.includes(f) ? 'var(--c-blue-fg)' : 'var(--text-secondary)',\n            fontSize: 10,\n            cursor: 'pointer'\n          }}\n        >\n          {f}\n        </button>\n      ))}\n    </div>\n  )\n}\n```\n\n### 우선순위 표시\n\n```tsx\nfunction PriorityChip({ priority }: { priority: 'low' | 'medium' | 'high' | 'urgent' }) {\n  const mapping = {\n    low: { tone: 'blue' as const, label: '낮음' },\n    medium: { tone: 'yellow' as const, label: '보통' },\n    high: { tone: 'orange' as const, label: '높음' },\n    urgent: { tone: 'red' as const, label: '긴급' }\n  }\n  \n  const { tone, label } = mapping[priority]\n  return <Chip tone={tone}>{label}</Chip>\n}\n```\n\n### 담당자 칩\n\n```tsx\nfunction AssigneeChip({ assignee }: { assignee: { name: string; color?: string } }) {\n  return (\n    <div style={{ display: 'flex', alignItems: 'center', gap: 4 }}>\n      <Avatar name={assignee.name} tone={assignee.color} size=\"sm\" />\n      <Chip tone=\"neutral\">{assignee.name}</Chip>\n    </div>\n  )\n}\n```\n\n## 인라인 메타\n\n여러 칩을 한 줄에 표시:\n\n```tsx\nfunction TaskMetadata({ task }: { task: Task }) {\n  return (\n    <div style={{ display: 'flex', gap: 8, alignItems: 'center', marginTop: 8 }}>\n      {task.status && <Chip tone=\"blue\">{task.status}</Chip>}\n      {task.priority && <Chip tone=\"orange\">{task.priority}</Chip>}\n      {task.assignee && (\n        <div style={{ display: 'flex', alignItems: 'center', gap: 4 }}>\n          <Avatar name={task.assignee} size=\"sm\" />\n          <Chip tone=\"neutral\">{task.assignee}</Chip>\n        </div>\n      )}\n      {task.dueDate && <Chip tone=\"yellow\"><Calendar size={10} /> {formatDate(task.dueDate)}</Chip>}\n    </div>\n  )\n}\n```\n\n## 외부 색 통합 (색-mix)\n\n사용자 정의 색(두레이 태그 등)을 칩으로 표시할 때:\n\n```tsx\nfunction UserTagChip({ tag }: { tag: { name: string; color: string } }) {\n  const bgColor = `color-mix(in oklab, ${tag.color} 18%, var(--bg-surface))`\n  const fgColor = tag.color\n  \n  return (\n    <span style={{\n      display: 'inline-flex',\n      alignItems: 'center',\n      gap: 4,\n      height: 18,\n      padding: '0 7px',\n      borderRadius: 999,\n      fontSize: 10,\n      fontWeight: 500,\n      background: bgColor,\n      color: fgColor,\n      whiteSpace: 'nowrap'\n    }}>\n      {tag.name}\n    </span>\n  )\n}\n```\n\n## 관련 토큰\n\n- `--c-*-bg`, `--c-*-fg`: 색상 쌍 (각 톤별)\n- `--text-secondary`: neutral 톤 텍스트\n- `--bg-border`: neutral 톤 테두리\n\n## 접근성\n\n칩은 시맨틱하지 않은 `<span>`이므로, 중요한 정보는 추가 마크업으로 표시:\n\n```tsx\n// 좋음: 상태를 aria-label로 명시\n<Chip tone=\"emerald\" title=\"작업이 완료됨\">완료</Chip>\n\n// 좋음: 목록에서 상태를 구분\n<div role=\"list\">\n  {tasks.map(t => (\n    <div key={t.id} role=\"listitem\">\n      <strong>{t.title}</strong>\n      <Chip tone={getStatusTone(t.status)}>{t.status}</Chip>\n    </div>\n  ))}\n</div>\n```\n\n## 자주 하는 실수\n\n### ✗ 너무 많은 칩\n\n```tsx\n// 안 됨 - 복잡함\n<div>\n  <Chip tone=\"blue\">tag1</Chip>\n  <Chip tone=\"orange\">tag2</Chip>\n  <Chip tone=\"red\">tag3</Chip>\n  <Chip tone=\"emerald\">tag4</Chip>\n  <Chip tone=\"violet\">tag5</Chip>\n  <Chip tone=\"yellow\">tag6</Chip>\n</div>\n\n// 좋음 - 주요 정보만\n<div>\n  <Chip tone=\"blue\">상태</Chip>\n  <Chip tone=\"orange\">우선순위</Chip>\n</div>\n```\n\n### ✗ 색상 남용\n\n```tsx\n// 안 됨 - 의미 없는 색상\n<Chip tone=\"violet\">기본값</Chip>\n<Chip tone=\"red\">기본값</Chip>\n\n// 좋음 - 의도 있는 색상\n<Chip tone=\"blue\">작업 중</Chip>\n<Chip tone=\"red\">오류</Chip>\n```\n\n## 예제 모음\n\n### 상태 배지\n\n```tsx\nfunction StatusBadge({ status }: { status: WorkflowStatus }) {\n  const tones: Record<WorkflowStatus, ChipTone> = {\n    backlog: 'neutral',\n    registered: 'orange',\n    working: 'blue',\n    resolved: 'emerald',\n    closed: 'neutral',\n    overdue: 'red'\n  }\n  \n  const labels: Record<WorkflowStatus, string> = {\n    backlog: '백로그',\n    registered: '등록됨',\n    working: '작업 중',\n    resolved: '해결됨',\n    closed: '종료',\n    overdue: '기한 초과'\n  }\n  \n  return (\n    <Chip tone={tones[status]} dot={status === 'working'}>\n      {labels[status]}\n    </Chip>\n  )\n}\n```\n
