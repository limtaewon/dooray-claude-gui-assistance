# StateViews\n\nStateViews는 빈 상태, 로딩 상태, 오류 상태를 표시하는 컴포넌트 모음입니다.\n\n## 컴포넌트\n\n### EmptyView\n\n데이터가 없을 때 표시합니다.\n\n#### Props\n\n```tsx\ninterface EmptyViewProps {\n  icon?: LucideIcon           // 아이콘 (기본: Inbox)\n  title: ReactNode            // 제목 (필수)\n  body?: ReactNode            // 설명 (선택)\n  action?: ReactNode          // 액션 버튼 (선택)\n}\n```\n\n#### 사용\n\n```tsx\nimport { EmptyView } from '@/components/common/ds/StateViews'\nimport { Inbox } from 'lucide-react'\n\nfunction TaskList() {\n  const tasks = useTasks()\n  \n  if (tasks.length === 0) {\n    return (\n      <EmptyView\n        icon={Inbox}\n        title=\"작업이 없습니다\"\n        body=\"새 작업을 만들어 시작하세요\"\n        action={<Button onClick={createTask}>새 작업</Button>}\n      />\n    )\n  }\n  \n  return <TaskGrid tasks={tasks} />\n}\n```\n\n#### 패턴\n\n```tsx\n// 기본\n<EmptyView title=\"결과 없음\" />\n\n// 커스텀 아이콘\n<EmptyView\n  icon={Search}\n  title=\"검색 결과 없음\"\n  body=\"다른 키워드로 검색해주세요\"\n/>\n\n// 액션 버튼\n<EmptyView\n  title=\"파일이 없습니다\"\n  action={<Button variant=\"primary\">파일 업로드</Button>}\n/>\n```\n\n### LoadingView\n\n데이터를 로드 중일 때 표시합니다.\n\n#### Props\n\n```tsx\ninterface LoadingViewProps {\n  label?: ReactNode  // 로딩 텍스트 (기본: \"불러오는 중...\")\n}\n```\n\n#### 사용\n\n```tsx\nimport { LoadingView } from '@/components/common/ds/StateViews'\n\nfunction DataContainer() {\n  const { data, loading } = useData()\n  \n  if (loading) {\n    return <LoadingView label=\"데이터 로드 중...\" />\n  }\n  \n  return <DataDisplay data={data} />\n}\n```\n\n#### 특징\n\n- 애니메이션 스피너 자동 표시\n- 텍스트 커스터마이징 가능\n- 전체 높이 채움\n\n### ErrorView\n\n오류 상태를 표시합니다.\n\n#### Props\n\n```tsx\ninterface ErrorViewProps {\n  title?: ReactNode        // 제목 (기본: \"문제가 발생했어요\")\n  body?: ReactNode         // 설명\n  onRetry?: () => void     // 재시도 핸들러 (있으면 버튼 표시)\n}\n```\n\n#### 사용\n\n```tsx\nimport { ErrorView } from '@/components/common/ds/StateViews'\n\nfunction DataFetcher() {\n  const { data, error, refetch } = useData()\n  \n  if (error) {\n    return (\n      <ErrorView\n        title=\"데이터 로드 실패\"\n        body=\"네트워크 연결을 확인해주세요\"\n        onRetry={refetch}\n      />\n    )\n  }\n  \n  return <DataDisplay data={data} />\n}\n```\n\n#### 패턴\n\n```tsx\n// 기본 오류\n<ErrorView />\n\n// 커스텀 메시지\n<ErrorView\n  title=\"페이지를 찾을 수 없습니다\"\n  body=\"요청한 페이지가 없거나 삭제되었을 수 있습니다\"\n/>\n\n// 재시도 옵션\n<ErrorView\n  title=\"연결 오류\"\n  body=\"서버에 연결할 수 없습니다\"\n  onRetry={() => location.reload()}\n/>\n```\n\n## 사용 패턴\n\n### 데이터 페칭 흐름\n\n```tsx\nfunction CompleteDataFlow() {\n  const { data, loading, error } = useData()\n  const [searchQuery, setSearchQuery] = useState('')\n  \n  // 로딩 중\n  if (loading) {\n    return <LoadingView label=\"데이터를 가져오는 중...\" />\n  }\n  \n  // 오류 발생\n  if (error) {\n    return (\n      <ErrorView\n        title=\"데이터 로드 실패\"\n        body={error.message}\n        onRetry={() => location.reload()}\n      />\n    )\n  }\n  \n  // 데이터 없음\n  if (!data || data.length === 0) {\n    return (\n      <EmptyView\n        title=\"데이터가 없습니다\"\n        body=\"새 항목을 만들어 시작하세요\"\n        action={<Button onClick={createItem}>새 항목</Button>}\n      />\n    )\n  }\n  \n  // 검색 결과 없음\n  const filtered = data.filter(item =>\n    item.name.includes(searchQuery)\n  )\n  \n  if (filtered.length === 0) {\n    return (\n      <EmptyView\n        title=\"검색 결과 없음\"\n        body={`\"${searchQuery}\"에 대한 결과가 없습니다`}\n      />\n    )\n  }\n  \n  // 정상 렌더링\n  return <DataGrid data={filtered} />\n}\n```\n\n### 리스트 페칭\n\n```tsx\nfunction TaskList() {\n  const { tasks, loading, error, retry } = useTasks()\n  \n  if (loading) return <LoadingView label=\"작업 불러오는 중...\" />\n  if (error) return <ErrorView onRetry={retry} />\n  if (tasks.length === 0) {\n    return (\n      <EmptyView\n        title=\"작업이 없습니다\"\n        action={<Button onClick={createTask}>새 작업</Button>}\n      />\n    )\n  }\n  \n  return <TaskList tasks={tasks} />\n}\n```\n\n### 모달 또는 컨테이너 여백\n\n## 아이콘 커스터마이징\n\n```tsx\nimport { AlertTriangle, Database, Lock } from 'lucide-react'\n\n// 검색 결과 없음\n<EmptyView icon={Search} title=\"검색 결과 없음\" />\n\n// 권한 없음\n<EmptyView icon={Lock}
