# Button

버튼 컴포넌트는 사용자 액션을 유도하는 기본 인터랙티브 요소입니다. 여러 스타일(variant)과 크기(size)를 제공하며, 아이콘과 텍스트를 함께 표시할 수 있습니다.\n\n## 언제 쓰나\n\n- **주요 액션 제출**: primary 버튼으로 \"저장\", \"생성\", \"전송\"\n- **보조 액션**: secondary 버튼으로 \"취소\", \"닫기\"\n- **위험 액션**: danger 버튼으로 \"삭제\", \"초기화\"\n- **AI 기능**: ai 버튼으로 Claude Code 연동 작업\n- **아이콘 전용**: icon 버튼으로 헤더 액션

## 언제 쓰면 안 되나\n\n- **단순 링크는 `<a>` 사용**: 페이지 이동은 `<Link>` 또는 `<a>`\n- **폼 제어는 `<Input>` 또는 `<SegTabs>` 사용**: 토글이나 선택 기능\n- **많은 텍스트**: 버튼 안에 2줄 이상 텍스트 금지\n\n## Props\n\n| 이름 | 타입 | 기본값 | 설명 |\n|------|------|--------|---|\n| `variant` | `'primary' \\| 'secondary' \\| 'ghost' \\| 'danger' \\| 'ai' \\| 'success' \\| 'orange' \\| 'icon'` | `'secondary'` | 스타일 |\n| `size` | `'xs' \\| 'sm' \\| 'md' \\| 'lg'` | `'md'` | 크기 |\n| `leftIcon` | `ReactNode` | — | 왼쪽 아이콘 |\n| `rightIcon` | `ReactNode` | — | 오른쪽 아이콘 |\n| `disabled` | `boolean` | — | 비활성화 |\n| `children` | `ReactNode` | — | 버튼 텍스트 |\n| `className` | `string` | — | 추가 CSS 클래스 |\n| `onClick` | `(e: MouseEvent) => void` | — | 클릭 핸들러 |\n| ...rest | `ButtonHTMLAttributes` | — | 표준 button 속성 |\n\n## Variant 가이드\n\n### Primary (기본 액션)\n\n```tsx\nimport Button from '@/components/common/ds/Button'\nimport { Plus } from 'lucide-react'\n\n<Button variant=\"primary\" size=\"md\" leftIcon={<Plus size={14} />}>\n  새 작업\n</Button>\n```\n\n**특징**:\n- 파랑 그래디언트 배경\n- 흰색 텍스트\n- 포커스 링 지원\n\n**용도**: 폼 제출, 주요 액션 (\"저장\", \"생성\", \"실행\")\n\n### Secondary (보조 액션)\n\n```tsx\n<Button variant=\"secondary\" size=\"md\">\n  취소\n</Button>\n```\n\n**특징**:\n- 배경색: 현재 surface 색\n- 테두리: 시스템 border\n- 텍스트: text-primary\n\n**용도**: 주요 액션을 보조하는 버튼 (\"취소\", \"닫기\")\n\n### Ghost (최소한 스타일)\n\n```tsx\n<Button variant=\"ghost\" leftIcon={<Share2 size={12} />}>\n  공유\n</Button>\n```\n\n**특징**:\n- 배경 없음\n- 텍스트는 secondary 색\n- 호버 시 surface-hover로 밝아짐\n\n**용도**: 부수적 액션 (\"더보기\", \"공유\"), 링크 같은 버튼\n\n### Danger (위험 액션)\n\n```tsx\n<Button variant=\"danger\" leftIcon={<Trash2 size={12} />}>\n  삭제\n</Button>\n```\n\n**특징**:\n- 텍스트는 red 색\n- 호버 시 red 배경\n\n**용도**: 삭제, 초기화, 취소 불가 액션. 주의 깊게 사용하세요.\n\n### Success (성공 상태)\n\n```tsx\n<Button variant=\"success\" leftIcon={<CheckCircle size={12} />}>\n  완료\n</Button>\n```\n\n**특징**:\n- 초록 배경 (`--c-emerald-solid`)\n- 흰색 텍스트\n\n**용도**: 성공 확인, 완료 표시. 드물게 사용합니다.\n\n### Orange (두레이 액션)\n\n```tsx\n<Button variant=\"orange\" leftIcon={<Send size={12} />}>\n  메시지 전송\n</Button>\n```\n\n**특징**:\n- 주황 배경 (`--c-orange-solid`)\n- 흰색 텍스트\n- 두레이 통합 작업 강조\n\n**용도**: 두레이 기능과 연동된 주요 액션\n\n### AI (AI 기능 강조)\n\n```tsx\n<Button variant=\"ai\" leftIcon={<Sparkles size={12} />}>\n  AI 분석\n</Button>\n```\n\n**특징**:\n- AI 그래디언트 배경 (주황 → 파랑)\n- 흰색 텍스트\n- 호버 시 투명도 감소\n\n**용도**: Claude Code 연동 작업, AI 기능 강조\n\n### Icon (아이콘 전용)\n\n```tsx\n<Button variant=\"icon\" size=\"sm\" aria-label=\"설정\">\n  <Settings size={16} />\n</Button>\n```\n\n**특징**:\n- 정사각형 (32×32px 기본)\n- 호버 시 surface-hover 배경\n- 아이콘만 표시\n\n**용도**: 헤더 액션 (\"닫기\", \"최소화\", \"설정\")\n\n## Size 가이드\n\n| Size | 높이 | padding | font-size | 용도 |\n|------|------|---------|-----------|------|\n| `xs` | 22px | 0 8px | 10.5px | 미니 액션 (tooltip) |\n| `sm` | 26px | 0 10px | 11px | 테이블 액션, 콤팩트 UI |\n| `md` | 32px | 0 12px | 12.5px | **기본** — 대부분의 UI |\n| `lg` | 36px | 0 14px | 13px | 대형 버튼 (모달 footer) |\n\n## 아이콘과 텍스트\n\n```tsx\n// 왼쪽 아이콘\n<Button\n  variant=\"primary\"\n  leftIcon={<Plus size={14} />}\n>\n  생성\n</Button>\n\n// 오른쪽 아이콘\n<Button\n  variant=\"secondary\"\n  rightIcon={<ChevronRight size={12} />}\n>\n  다음\n</Button>\n\n// 양쪽 아이콘 (드물게)\n<Button\n  leftIcon={<Send size={12} />}\n  rightIcon={<ExternalLink size={12} />}\n>\n  전송\n</Button>\n\n// 아이콘만\n<Button variant=\"icon\" size=\"sm\">\n  <Bell size={14} />\n</Button>\n```\n\n**가이드**:\n- 아이콘 크기는 보통 12-14px\n- gap은 CSS에서 자동 적용 (6px)\n- 텍스트가 짧으면 아이콘이 더 눈에 띕니다\n\n## 비활성화 (Disabled)\n\n```tsx\n<Button variant=\"primary\" disabled>\n  저장\n</Button>\n\n// 또는\n<Button variant=\"primary\" aria-disabled=\"true\">\n  로딩 중...\n</Button>\n```\n\n**특징**:\n- 투명도 45%\n- cursor: not-allowed\n- 호버/클릭 반응 없음\n\n## 로딩 상태\n\nbutton 내부에 로딩 스피너를 직접 넣으세요:\n\n```tsx\nconst [loading, setLoading] = useState(false)\n\n<Button\n  variant=\"primary\"\n  disabled={loading}\n  onClick={async () => {\n    setLoading(true)\n    try {\n      await saveData()\n    } finally {\n      setLoading(false)\n    }\n  }}\n>\n  {loading ? (\n    <>\n      <div className=\"ds-spinner\" style={{ width: 12, height: 12 }} />\n      저장 중...\n    </>\n  ) : (\n    '저장'\n  )}\n</Button>\n```\n\n## 접근성\n\n### 키보드 네비게이션\n\n- Tab 키로 포커스 이동\n- Enter/Space로 활성화\n- 포커스 링이 자동 표시됨\n\n### 스크린 리더\n\n```tsx\n// 좋음: 텍스트가 명확\n<Button variant=\"primary\">삭제</Button>\n\n// 아이콘만 있을 때는 aria-label 필수\n<Button variant=\"icon\" aria-label=\"설정 열기\">\n  <Settings size={14} />\n</Button>\n\n// 로딩 상태 표시\n<Button disabled aria-busy=\"true\">\n  저장 중...\n</Button>\n```\n\n### 색상 대비\n\n모든 variant는 WCAG AA 표준(4.5:1 이상)을 만족합니다.\n\n## 관련 토큰\n\n- `--c-blue-solid`: Primary 배경\n- `--c-red-fg`, `--c-red-bg`: Danger 버튼\n- `--c-emerald-solid`: Success 버튼\n- `--c-orange-solid`: Orange/AI 버튼\n- `--text-primary`, `--text-secondary`: 텍스트 색\n- `--ring-focus`: 포커스 링\n- `--bg-border`: Secondary 테두리\n\n## 자주 하는 실수\n\n### ✗ 텍스트가 너무 김\n\n```tsx\n// 안 됨\n<Button>이 작업을 실행하시겠습니까? 확인하세요</Button>\n\n// 좋음\n<Button>실행</Button>\n// 긴 설명은 모달/알림에 넣기\n```\n\n### ✗ 너무 많은 variant\n\n```tsx\n// 안 됨 - 복잡함\n<div>\n  <Button variant=\"primary\">저장</Button>\n  <Button variant=\"secondary\">취소</Button>\n  <Button variant=\"danger\">삭제</Button>\n  <Button variant=\"ghost\">더보기</Button>\n</div>\n\n// 좋음 - 명확한 계층\n<div>\n  <Button variant=\"primary\">저장</Button>\n  <Button variant=\"secondary\">취소</Button>\n</div>\n<Button variant=\"ghost\" size=\"sm\">고급 옵션</Button>\n```\n\n### ✗ Disabled 남용\n\n```tsx\n// 안 됨 - 이유를 알 수 없음\n<Button disabled>저장</Button>\n\n// 좋음 - 툴팁이나 텍스트로 이유 표시\n<Button disabled title=\"모든 필드를 입력해야 합니다\">\n  저장\n</Button>\n```\n\n## 예제 모음\n\n### 폼 액션\n\n```tsx\nimport Button from '@/components/common/ds/Button'\nimport { Save, X } from 'lucide-react'\n\nfunction SaveForm() {\n  return (\n    <div style={{ display: 'flex', gap: 8 }}>\n      <Button\n        variant=\"primary\"\n        leftIcon={<Save size={12} />}\n        onClick={handleSave}\n      >\n        저장\n      </Button>\n      <Button variant=\"secondary\" leftIcon={<X size={12} />}>\n        취소\n      </Button>\n    </div>\n  )\n}\n```\n\n### 헤더 액션\n\n```tsx\nfunction PageHeader() {\n  return (\n    <div style={{ display: 'flex', gap: 4 }}>\n      <Button variant=\"icon\" size=\"sm\" aria-label=\"뒤로\">\n        <ChevronLeft size={16} />\n      </Button>\n      <Button variant=\"icon\" size=\"sm\" aria-label=\"설정\">\n        <Settings size={16} />\n      </Button>\n    </div>\n  )\n}\n```\n\n### AI 작업\n\n```tsx\nfunction AIDashboard() {\n  return (\n    <Button\n      variant=\"ai\"\n      leftIcon={<Sparkles size={14} />}\n      onClick={runAnalysis}\n    >\n      분석 실행\n    </Button>\n  )\n}\n```\n
