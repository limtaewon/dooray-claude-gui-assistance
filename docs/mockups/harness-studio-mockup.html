blocks/ — 재사용 스크립트·패턴 (여러 에이전트가 공유 · 중복 제거)

게이트 & 강제 — "~하지 마세요"는 강제가 아니다. 강제는 코드가 차단(exit 2)할 때만 강제다. 4계층 제약 중 런타임 강제는 둘뿐 — 도구 권한 + 자동 hook.

4계층 제약 — 무엇이 진짜 강제인가

gate-check.sh — 페이즈 전환 게이트 (exit 0 통과 / exit 2 차단 · 우회 NEON_BMAD_SKIP_RULES)

자동 hook — LLM 협조 없이 차단

pipeline.sh — 자율 루프 상태기계 ("고삐는 멈추지 않는다")

advance = 게이트를 직접 호출해 exit code·verdict로 전이 결정 — LLM이 결과를 위조할 수 없는 폐루프. 루프 지속/종료는 Stop hook.
neon-bmad는 스스로를 6축으로 적대적 채점(harness-critic, Opus)합니다. 각 축 = 골격이 행동을 강제·기록(2) / 모델 협조 기대(1) / 없음(0). "프롬프트 더미"와 "하네스"를 가르는 기준.
